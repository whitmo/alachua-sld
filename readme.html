Data

Alachua's data is stored in a folder of shapefiles that are updated daily but a script. We have suggested that they switch to PostGIS, but at this point they are unwilling to make the change. As a result we need to configure GeoServer to read and server this data dir.

Location

Its localtion on the www.nikos.alachua.fl.us server is:

C:/ArcIMS/Data/*

The sub-folders are as follows:

GeoCEData
GeoDRDdata
GeoGMData
MultiPatch
PermitData
